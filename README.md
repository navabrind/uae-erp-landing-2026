# Navabrind UAE — Landing Page

Live site: https://uae.navabrindsol.com

## Structure

```
index.html   ← Main page (single-file, all CSS + JS inline)
logo.png     ← Navabrind logo (greyscale PNG, 640×195px)
netlify.toml ← Netlify build config
```

## How to edit

1. Edit `index.html` directly — all styles are inline in `<style>` and all scripts are at the bottom of the file.
2. To update the logo, replace `logo.png` (keep the same filename).
3. Commit and push to `main` → Netlify auto-deploys to https://uae.navabrindsol.com within ~30 seconds.

## Deploy

Hosted on Netlify, connected to this repo. Every push to `main` triggers an automatic production deploy.

- **Netlify site**: `uae-navabrindsol`
- **Custom domain**: `uae.navabrindsol.com`
- **Netlify dashboard**: https://app.netlify.com/projects/uae-navabrindsol
